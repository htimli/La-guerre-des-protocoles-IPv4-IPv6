Detailed Description

An implementation of the Coca project for 2020 year. Converts a 2-coloured graph g and an integer p to a formula true if and only if it exists a translator set of size p. Provides functions to generate the formula and the necessary variables, alongside function to decode a solution from a model of the formula.
